# Analytics Privacy Policy V1

## Status

Approved

---

# Purpose

The Analytics Privacy Policy defines the privacy boundaries of the Minime Analytics System.

Its purpose is to ensure that analytics remain useful while minimizing the collection of personal information.

---

# Core Principle

```text
Measure Activity
Not People
```

Analytics V1 measures profile activity.

Analytics V1 does not identify individuals.

---

# Privacy Philosophy

Minime Analytics is designed to answer:

```text
What happened?
```

not:

```text
Who did it?
```

---

# Data Minimization

Analytics V1 collects only the information required to calculate approved metrics.

Any data not required for:

```text
Views
Clicks
CTR
Device Category Breakdown
```

should not be collected.

---

# Supported Analytics Data

Analytics V1 may use:

```text
Profile View Events
Link Click Events
Device Category
Timestamp
```

Only.

---

# Device Information

Supported device categories:

```text
Desktop
Mobile
Tablet
```

Only.

Analytics does not collect:

```text
Device Model
Manufacturer
Hardware Information
```

---

# Visitor Identification

Analytics V1 does not identify visitors.

Not supported:

```text
User Identity
Visitor Identity
Account Linking
Cross Profile Tracking
```

---

# Unique Visitor Tracking

Analytics V1 does not calculate:

```text
Unique Visitors
```

Reason:

```text
No Visitor Identification Exists
```

---

# Sessions

Analytics V1 does not create:

```text
Sessions
Session IDs
Session Histories
```

Not supported.

---

# Cookies

Analytics V1 does not require:

```text
Analytics Cookies
Tracking Cookies
Visitor Cookies
```

Not supported.

---

# Fingerprinting

Analytics V1 does not perform:

```text
Browser Fingerprinting
Device Fingerprinting
Visitor Fingerprinting
```

Not supported.

---

# Location Tracking

Analytics V1 does not collect:

```text
Country
Region
City
Coordinates
Location History
```

Not supported.

---

# Browser Tracking

Analytics V1 does not collect:

```text
Browser Name
Browser Version
Browser History
```

Not supported.

---

# Operating System Tracking

Analytics V1 does not collect:

```text
Windows
macOS
Linux
Android
iOS
```

information.

Not supported.

---

# Referrer Tracking

Analytics V1 does not collect:

```text
Referrer URLs
Traffic Sources
Campaign Sources
```

Not supported.

---

# Behavioral Tracking

Analytics V1 does not collect:

```text
User Journeys
Behavior Flows
Scroll Tracking
Interaction Tracking
Heatmaps
```

Not supported.

---

# Advertising Use

Analytics data must not be used for:

```text
Advertising Profiles
Audience Building
Behavioral Advertising
Retargeting
```

Not supported.

---

# Analytics Access

Analytics data is available only to:

```text
Account Owner
```

and authorized system administrators.

Analytics data is not public.

---

# Data Ownership

Analytics data belongs to:

```text
The Account Owner
```

and is associated with:

```text
The Account
```

---

# Reporting Restrictions

Reports may display:

```text
Counts
Percentages
Rankings
Breakdowns
```

Only.

Reports must not expose:

```text
Visitor Information
Personal Information
Individual Activity
```

---

# Event Integrity

Analytics events represent:

```text
System Activity
```

not personal identities.

Events must remain anonymous.

---

# Privacy Boundary

Analytics V1 intentionally stops at:

```text
Activity Measurement
```

and does not expand into:

```text
Audience Intelligence
Visitor Profiling
Behavior Analysis
```

---

# V1 Non-Goals

Not supported:

```text
Marketing Attribution
Audience Segmentation
Remarketing
Identity Resolution
User Profiling
Cross Site Tracking
```

---

# Design Goal

The Analytics System should remain:

```text
Privacy-Light
Transparent
Minimal
Respectful
```

while still providing useful insights through:

```text
Views
Clicks
CTR
Device Category Breakdown
```

and nothing more.
